VidCrop 是一款视频裁剪工具，帮助用户轻松裁剪视频文件。您可以手动选择视频的任意部分进行裁剪，程序内置视频播放器，方便在裁剪前后查看预览。

## 主要功能

- 支持 .mp4, .m4v, .mov, .mpg, .mpeg, .3gp 等常见视频格式。
- 用户可以通过拖动调整裁剪框或自定义视频尺寸，精确裁剪视频的任意部分。
